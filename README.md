# Symbol Removing for File Management

A lightweight desktop utility for cleaning file names by removing or replacing unwanted symbols.

Designed for photographers, office users, and anyone managing large structured folders.

---

## ✨ What It Does

Batch process files and:

- Remove selected symbols from filenames
- Replace characters with custom text
- Keep structure clean and readable
- Work across folders and subfolders

No complicated setup. Just drag, choose symbols, apply.

---

## 🚀 Features

- Drag & drop support
- Batch processing
- Subfolder option
- Safe renaming logic
- Clean and simple GUI
- Preview before applying changes

---

## 💡 Why Use It?

File names coming from:

- Cameras
- Export tools
- Messaging apps
- Downloads

Often contain unwanted symbols like:
% & ( ) [ ] { } @ ! + =

This tool cleans them fast and consistently.
